# NewsroomOps UI

A frontend-focused newsroom content operations dashboard designed to simulate real-world editorial workflows.

This project demonstrates how internal newsroom tools handle article review, scheduling, publishing, and audit tracking — with an emphasis on frontend architecture and state management.

---

## Features

- Content queue with pagination and filters
- Article detail drawer with metadata and quality checks
- Editorial workflow actions (Draft → Review → Scheduled → Published)
- Schedule picker with date/time validation
- Audit log with field-level diffs (before → after)
- Optimistic UI updates using React Query
- Mocked API layer using Mock Service Worker (MSW)

---

## Tech Stack

- **React + TypeScript**
- **Vite**
- **Tailwind CSS**
- **React Query**
- **Mock Service Worker (MSW)**

---

## Notes

This project uses MSW to simulate backend APIs for demonstration purposes.
The focus is entirely on frontend behavior, UX, and state management patterns commonly used in internal tools.
